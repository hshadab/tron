The Problem

AI agents can spend money.
Who's watching?

Autonomous agents make payments without human approval. A single prompt injection can drain a treasury.

Preflight interposes a policy layer: every payment is verified by 3 independent solvers and sealed with a ZK proof before any funds move.

Settlement happens on TRON via the x402 payment protocol — only if all solvers agree the action is policy-compliant.
